问题01

一个可运行的C++源程序( )。

A. 由一个或多个主函数构成
B. 由一个且仅由一个主函数和零个以上（含零个）的子函数构成
C. 仅由一个主函数构成
D. 由一个且只有一个主函数和多个子函数构成

答案: B

解释:

在C++程序中，一个可运行的程序必须包含一个且仅有一个main函数，因为main函数是程序的入口点。没有main函数的程序是无法运行的。然而，除了main函数之外，程序还可以包含零个或多个其他的子函数，这些子函数可以在main函数中被调用或作为其他函数的辅助功能。因此，选项B是正确的，描述了一个C++可运行程序的基本结构。

  • A不正确，因为一个程序只能有一个main函数。
  • C不完全正确，因为虽然程序至少有一个main函数，但它可以包含其他子函数。
  • D不完全正确，因为程序可以包含零个子函数。

问题02

C++程序的三种基本结构是( )。

A. 顺序结构、选择结构、循环结构
B. 递归结构、循环结构、转移结构
C. 嵌套结构、递归结构、顺序结构
D. 循环结构、转移结构、顺序结构

答案: A

解释:

C++程序（以及大多数编程语言）的基本控制结构包括顺序结构、选择结构和循环结构。这三种结构是程序控制流程的基本方式：

  1. 顺序结构：程序按顺序执行每条语句，直到结束。
  2. 选择结构：程序根据条件执行不同的分支，例如if-else和switch语句。
  3. 循环结构：程序重复执行某段代码，例如for、while和do-while循环。

选项A正确描述了这三种基本结构：

  • B不正确，因为递归不是一种基本结构，而是函数调用的一种形式。
  • C不正确，因为嵌套和递归不是基本结构，而是代码组织方式。
  • D不正确，因为转移结构（如goto语句）并不是基本结构，且在现代编程中不常推荐使用。

问题03

以下不属于字符型常量的是（ ）。

A． ‘$’
B． ‘+’
C． ‘x2A’
D． “B”

答案: D

解释:

在C++中，字符型常量是单个字符用单引号括起来的，例如 'a', '1', '$'。字符型常量的本质是一个单独的字符，存储为一个char类型。

  • A ‘$’ 是一个字符型常量，表示字符$。
  • B ‘+’ 是一个字符型常量，表示字符+。
  • C ‘x2A’ 是一个字符型常量，表示字符x，但这里应该注意写法是否有误。如果是 'x' 才是有效字符型常量，而 'x2A' 在单引号中多于一个字符会导致语法错误。但根据选项，主要的错误还是 D。
  • D “B” 是一个字符串常量，而不是字符型常量。字符串常量用双引号括起来，可以包含多个字符，即使是单个字符用双引号括起来也是字符串类型，而不是字符型常量。

因此，D是正确答案，因为它表示的是字符串常量，不是字符型常量。

问题04

下列变量定义中, 错误的是( )。

A． int if;
B． int a,A;
C． char a=’’;
D． float x=-3.33333;

答案: A

解释:

  • A int if; 是错误的，因为if是C++中的保留字，用于条件语句，不能用作变量名。
  • B int a,A; 是正确的，C++区分大小写，所以a和A是两个不同的变量。
  • C char a=’’; 是错误的，因为字符常量应该包含一个字符，空字符常量（即没有字符）是无效的。正确的定义应该是 char a=' '; (表示空格字符)或者 char a='\0'; (表示空字符)。
  • D float x=-3.33333; 是正确的，定义了一个浮点型变量x并赋值为-3.33333。

尽管C也是错误的，但根据选项描述，最明显错误的选项是A，因为它直接违反了C++语法规则，即不能使用关键字作为变量名。因此答案是A。

问题05

已知：char a; float b; double c; 则执行语句：c = a + b + c; 后变量c的类型为( )。

A．char
B．int
C．float
D．double

答案: D

解释:

在C++中，当进行算术运算时，数据类型的转换遵循一定的规则（隐式类型转换）。这些规则决定了不同类型的数据在运算过程中会被转换为哪种类型，以保证运算的精度和正确性。

具体规则包括：

  1. 所有char和short类型会被提升为int。
  2. 如果运算中包含float和double，则结果会转换为double类型。

在语句c = a + b + c;中：

  • a是char类型，会被提升为int。
  • b是float类型。
  • c是double类型。

由于double具有最高的精度，在表达式中a + b会被转换为float，然后再加上c时，整个表达式会被提升为double。最终，表达式的结果是double类型，并赋值给c，c的类型仍然是double。

因此，答案是D。

问题06

设已有说明语句int a=5;，则执行语句a+=a*=a%=3;后，a的值为( ) 。

A．8
B．18
C．0
D．2

答案: A

解释:

在C++中，赋值运算符的结合性是从右向左。因此我们需要从右向左依次解析和执行运算符。

首先来看具体的操作：

  1. a %= 3：

    • a的初始值是5。
    • 5 % 3 等于 2。
    • 所以a变为2。

  2. a *= a：

    • 现在a的值是2。
    • 2 * 2 等于4。
    • 所以a变为4。

  3. a += a：

    • 现在a的值是4。
    • 4 + 4 等于8。
    • 所以a最终变为8。

因此，执行语句a+=a*=a%=3;后，a的值是8。

答案是A。

问题07

对数组名作函数的参数，下面描述正确的是（　）。

A．数组名作函数的参数，调用时将实参数组复制给形参数组
B．数组名作函数的参数，主调函数和被调函数共用一段存储单元
C．数组名作参数时，形参定义的数组长度不能省略
D．数组名作参数，不能改变主调函数中的数据

答案: B

解释:

在C++中，当数组名作为函数参数传递时，传递的是数组的指针。这意味着在主调函数和被调函数中，数组名实际上指向同一段内存地址，因此两者共用同一段存储单元。这就允许在被调函数中修改数组内容，并在主调函数中看到这些修改。

  • A不正确，因为数组名作为参数时传递的是指针，而不是复制整个数组。
  • B正确，因为数组名作为参数传递时，主调函数和被调函数确实共用一段存储单元。
  • C不正确，因为在函数形参中，数组长度可以省略。例如，void func(int arr[])是合法的。
  • D不正确，因为通过数组指针可以改变主调函数中的数据。

问题08

给出下面定义：

则下列说法正确的是（ ）。

A. 数组a与数组b等价
B. 数组a和数组b的长度相同
C. 数组a的长度大于数组b的长度
D. 数组a的长度小于数组b的长度

答案: C

解释:

  1. 数组a的定义:

    这里，a是一个字符串常量，字符串以null字符（'\0'）结尾。因此，数组a的长度是5，包括'a', 'b', 'c', 'd', 和'\0'。

  2. 数组b的定义:

    这里，b是一个字符数组，不是字符串，它的长度是4，不包含null字符。

因此：

  • A不正确，因为a是一个包含终止符的字符串，而b只是一个字符数组。
  • B不正确，因为a的长度是5，而b的长度是4。
  • C正确，因为数组a的长度（5）大于数组b的长度（4）。
  • D不正确，因为数组a的长度并不小于数组b的长度。

关于strlen和sizeof的解释：

  • strlen函数用于求字符串的长度，不包括终止符'\0'。
  • sizeof运算符用于求数组在内存中的长度，包括终止符'\0'。

因此在sizeof(a)中会包括'\0'，而sizeof(b)不会。

问题14

B. x[a]

答案: D

解释:

在C++中，使用.运算符来访问对象的成员。对于一个类的对象x，使用.运算符加上成员名a即可访问x对象中的公有数据成员a。因此，选项D是正确的。

所以，正确答案是D。

答案: D

解释:

函数原型（或者叫函数声明）主要用于告诉编译器函数的接口，包括函数的返回类型、函数名、以及参数列表。但是，函数原型通常不包含函数的具体实现或功能。因此，函数原型不能标识函数的具体功能，选项D是正确的。

问题 33

使用地址作为实参传给形参，下列说法正确的是（ ）。

A. 实参是形参的备份
B. 实参与形参无联系
C. 形参是实参的备份
D. 实参与形参是同一对象

答案: D

解释:

在C++中，如果使用地址作为实参传递给形参，那么形参将指向实参的地址。因此，形参和实参实际上指向内存中的同一块区域，它们是同一个对象的不同名称。因此，选项D是正确的。

问题 34

下列不是描述类的成员函数的是（ ）。

A. 构造函数
B. 析构函数
C. 友元函数
D. 拷贝构造函数

答案: C

解释:

类的成员函数包括构造函数、析构函数、以及其他成员函数。构造函数用于初始化对象的数据成员，析构函数用于清理对象。拷贝构造函数用于创建一个对象的副本。而友元函数虽然与类相关，但它不是类的成员函数，因为它不属于类的作用域，而是在类的外部定义的函数。

问题 35

下面有关类说法不正确的是（ ）。

A. 一个类可以有多个构造函数
B. 一个类只有一个析构函数
C. 析构函数需要指定参数
D. 在一个类中可以说明具有类类型的数据成员

答案: C

解释:

选项C描述不正确。析构函数不需要指定参数。析构函数的语法是在类名前加上波浪号（~），并且没有参数列表。它用于释放对象使用的资源，在对象被销毁时自动调用。

问题 36

在函数定义中的形参属于（ ）。

A. 全局变量
B. 局部变量
C. 静态变量
D. 寄存器变量

答案: B

解释:

形参属于函数的局部变量，在函数被调用时才会被创建，在函数执行完毕后会被销毁。它们通常存储在栈上，因此属于局部变量的范畴。

问题 37

下列有关重载函数的说法中正确的是（ ）。

A. 重载函数必须具有不同的返回值类型
B. 重载函数参数个数必须相同
C. 重载函数必须有不同的形参列表
D. 重载函数名可以不同

答案: C

解释:

重载函数是指在同一个作用域内，可以有多个函数名相同但参数列表不同的函数。因此，选项C是正确的。重载函数的参数列表必须不同，可以是参数个数不同、参数类型不同、或者参数顺序不同。其余选项描述是错误的：

  • A 重载函数可以具有相同的返回值类型，因为函数的重载是通过参数列表来区分的，而不是通过返回值类型。
  • B 重载函数的参数个数可以不同。
  • D 重载函数名必须相同，这是重载函数的定义特点之一。

问题 38

this指针存在的目的是（ ）。

A. 保证基类私有成员在子类中可以被访问
B. 保证基类保护成员在子类中可以被访问
C. 保证每个对象拥有自己的数据成员，但共享处理这些数据成员的代码
D. 保证基类公有成员在子类中可以被访问

答案: C

解释:

this指针是一个指向当前对象的指针，在成员函数中使用。它的主要目的是让每个对象都能够访问自己的数据成员，而不论这些数据成员是否是静态的。它并不涉及基类与子类的访问权限问题，也不会影响到继承体系中的访问控制。

问题 39

以下基类中的成员函数表示纯虚函数的是（ ）。

A. virtual void tt()=0
B. void tt(int)=0
C. virtual void tt(int)
D. virtual void tt(int){}

答案: A

解释:

纯虚函数是一种在基类中声明但没有实现的虚函数，它的目的是使得该类成为抽象类，不能被实例化，只能作为其他类的基类。在C++中，我们可以通过在函数声明的末尾加上=0来声明纯虚函数，例如选项A中的virtual void tt()=0。这表示tt()是一个纯虚函数，子类必须实现它，否则子类也会成为抽象类。其余选项不是纯虚函数的声明方式。

A.内联函数

B.重载函数

C.递归函数

D.函数模板

答案

A

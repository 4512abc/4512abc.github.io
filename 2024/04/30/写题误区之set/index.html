写题遇到的Set误区

C++ set rbegin() 函数用于返回引用集合容器最后一个元素的反向迭代器。

set 的反向迭代器以相反的方向移动并递增它，直到它到达 set 容器的开头(第一个元素)。

用法

参数

空

返回值

它返回一个反向迭代器(反向迭代器)，它指向集合的最后一个元素。

复杂度

恒定。

迭代器有效性

没有变化。

数据竞争

非常量和常量版本都不会访问集合，不会修改集合容器。同时访问集合的元素是安全的。

异常安全

此函数从不抛出异常。

输出：

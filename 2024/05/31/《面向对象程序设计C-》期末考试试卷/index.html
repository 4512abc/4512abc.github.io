《面向对象程序设计C++》期末考试试卷

一、选择题(每小题3分，共48分。)

01、下列关于C++函数的叙述中，正确的是( C )。

A)每个函数至少要具有一个参数 B)每个函数都必须返回一个值

C)函数在被调用之前必须先声明 D)函数不能自己调用自己

02、下列关于类和对象的叙述中，错误的是( A )。

A)一个类只能有一个对象 B)对象是类的具体实例

C)类是对某一类对象的抽象 D)类和对象的关系是一种数据类型与变量的关系

01、正确答案是 C) 函数在被调用之前必须先声明。在 C++ 中，函数可以在调用之前进行声明，也可以在调用之后进行定义。但在调用之前，编译器需要知道函数的声明或原型，以便正确地解析函数调用。

02、错误答案是 A) 一个类只能有一个对象。在 C++ 中，一个类可以有多个对象实例化。

03、在C++中，用于实现运行时多态性的是( D )。

A)内联函数 B)重载函数 C)模板函数 D)虚函数

04、下列关于[运算符]重载的叙述中，正确的是( B )。

A)通过运算符重载，可以定义新的运算符 B)有的运算符只能作为成员函数重载

C)若重载运算符+，则相应的运算符函数名是+ D)重载二元运算符时，必须声明两个形参

03、正确答案是 D) 虚函数。在 C++ 中，通过使用虚函数和继承，可以实现运行时多态性。

04、正确答案是 B) 有的运算符只能作为成员函数重载。C++ 中有一些运算符只能被重载为类的成员函数，例如赋值运算符 = 和下标运算符 []。

05、对于语句 cout<<x<<endl;错误的是描述是( D )。

A)“cout”是一个输出流对象 B)“endl”的作用是输出回车换行

C)“x”是一个变量 D)“<<”称作提取运算符

D) “<<” 称为插入运算符，不是提取运算符。它用于将数据插入到输出流中，而不是提取数据。
这个是插入运算符

06、有如下类声明，则类MyDERIVED中保护的数据成员和成员函数的个数是( B )。

　A)4 B)3 C)2 D)1

在类 MyDERIVED 中，保护的数据成员和成员函数的个数是 3。这是因为：

  1. 成员变量 j 是保护的（protected）。
  2. 成员函数 set(int m, int n) 是公有的（public），它调用了基类 MyBASE 的 set(int n) 函数，但 MyBASE::set(int n) 在基类中是私有的（private）。
  3. 成员函数 get() const 是公有的（public），它调用了基类 MyBASE 的 get() 函数，但 MyBASE::get() 在基类中是公有的（public）。

因此，除去派生类 MyDERIVED 自己新增的成员之外，保护的数据成员和成员函数总共有 3 个。

07、已知在一个类体中包含如下函数原型: Volume operator-(Volume)const;，下列关于这个函数的叙述中，错误的是( B)。

A)这是运算符-的重载运算符函数 　B)这个函数所重载的运算符是一个一元运算符

C)这是一个成员函数 　D)这个函数不改变类的任何数据成员的值

08、执行如下的程序段后，输出结果是( )。

cout.fill(‘*’); cout.width(6); cout.fill(‘#’); cout << 123 << endl;

A)###123 B)123### C)***123 D)123***

07、错误的叙述是 B) 这个函数所重载的运算符是一个一元运算符。实际上，这个函数所重载的是减法运算符 -，但它是一个双目运算符，因为它需要两个操作数。

08、执行如下程序段后，输出结果是：C) ###123。

09、在下列函数原型中，可以作为类AA构造函数的是( D )。

A)void AA(int); B)int AA(); C)AA(int)const; D)AA(int);

10、下面四个选项中，( A )是用来声明虚函数的。

A)virtual B)public C)include D)using namespace

09、可以作为类 AA 构造函数的是 D) AA(int)。构造函数的名称必须与类名完全相同，不带任何返回类型，因此只有 D 选项符合这一要求。

10、用来声明虚函数的是 A) virtual。在 C++ 中，通过在函数声明或定义前加上 virtual 关键字，可以将该函数声明为虚函数，以实现运行时多态性。

11、实现运行时的多态性要使用( D )。

A)重载函数 B)构造函数 C)析构函数 D)虚函数

12、要实现动态联编，必须通过( A )调用虚函数。

A)对象指针 B)成员名限定 C)对象名 D)派生类名

11、实现运行时的多态性需要使用 D) 虚函数。通过在基类中声明虚函数，然后在派生类中进行覆盖，可以在运行时动态地选择调用相应的派生类函数。

12、要实现动态联编，必须通过 A) 对象指针 调用虚函数。在 C++ 中，如果想要实现动态联编（即在运行时确定调用哪个函数），通常需要使用基类的指针或引用来调用虚函数。这样可以根据指针或引用所指向的实际对象类型来决定调用哪个函数，而不是根据指针或引用的静态类型来决定。

13、以下( C )成员函数表示纯虚函数。

A)virtual int vf(int); B)void vf(int)=0;

C)virtual void vf()=0; D)virtual void vf(int){}

正确答案是 C) virtual void vf()=0;。这个成员函数是一个纯虚函数，通过在函数声明的结尾处使用 =0 来声明。纯虚函数是在基类中声明的虚函数，但没有在基类中给出具体的实现，而是留待派生类去实现。这使得基类成为了一个抽象类，不能被实例化。

14、有如下类声明：

在构造函数XB的下列定义中，正确的是( B )。

A)XB::XB(int a,int b): x(a),y(b){ } B)XB::XB(int a,int b): XA(a),y(b){ }

C)XB::XB(int a,int b): x(a),XB(b){ } D)XB::XB(int a,int b): XA(a),XB(b){ }

正确的定义是 B) XB::XB(int a, int b): XA(a), y(b) { }。

在派生类的构造函数中，首先要调用基类的构造函数来初始化基类的部分。在这里，使用 XA(a) 来调用基类的构造函数。然后，初始化派生类新增的成员变量 y，可以直接使用初始化列表 y(b)。

15、在表达式 x+$yz$中， + 是作为成员函数重载的运算符， 是作为非成员函数重载的运算符。则 operator+ 有 个参数，operator* 有 参数。( C )

A)2、2 B)2、1 C)1、2 D)1、1

16、应在下列程序划线处填入的正确语句是( C )。

15、正确答案是 C) 1、2。

  • operator+ 作为成员函数重载，因此它有一个参数，即另一个操作数。
  • operator* 作为非成员函数重载，因此它有两个参数，分别是两个操作数。

16、应在下列程序划线处填入的正确语句是 C) Base::fun();。

在派生类中，如果想要显式调用基类的同名函数，应该使用作用域解析运算符 :: 来指明基类的名称和函数名。

二、填空题(每空2分，共20分。)

17、面向对象程序设计有四个主要特点，即抽象、封装、继承和多态性。

18、非成员函数应声明为类的_friend___函数才能访问这个类的private成员。

19、派生类中的成员不能直接访问基类中的private成员。

20、在用class定义一个类时，数据成员和成员函数的默认访问权限是_private___。

21、运算符重载函数可能是类的 友元函数，也可以是类的_成员_函数，还可以是普通函数。

22、用流对象的成员函数控制输出格式时，用于设置字段宽度的流成员函数的名称是，与之作用相同的控制符名称是。

23、含有纯虚函数的类称为抽象类。

17、面向对象程序设计的四个主要特点是抽象、封装、继承和多态性。这些特点是面向对象编程的基础，通过它们可以构建出灵活、可维护、可扩展的程序。

18、非成员函数如果需要访问类的私有成员，可以声明为类的友元函数。这样做会允许该函数访问类的私有成员，但同时也会破坏封装性，因此需要谨慎使用。

19、派生类中的成员函数可以访问基类中的受保护成员，但不能直接访问基类中的私有成员。私有成员只能在其所在的类中访问，无法在派生类中直接访问。

20、在用 class 定义一个类时，数据成员和成员函数的默认访问权限是 private。这意味着在 class 中声明的数据成员和成员函数默认情况下都是私有的，只能在类内部访问。

21、运算符重载函数可以是类的友元函数，也可以是类的成员函数，还可以是普通函数。这取决于具体的设计需求和实现方式。

22、用于设置字段宽度的流成员函数的名称是 width()，与之作用相同的控制符名称是 setw()。width() 是流对象的成员函数，用于设置字段宽度；而 setw() 是流操作符 setw，用于临时设置字段宽度。

23、含有纯虚函数的类称为抽象类。抽象类是不能被实例化的，因为它包含了至少一个纯虚函数，这意味着它的接口没有具体的实现。抽象类主要用于定义接口，让派生类去实现具体的功能。

a)抽象出一个基类base；b)在其中说明一个虚函数用来求面积；c)利用派生类定义“三角形、正方形、圆形”；d)编写主函数并测试。

这个程序定义了一个基类 Shape，其中声明了一个纯虚函数 getArea() 用于求面积。然后分别定义了三个派生类 Triangle、Square 和 Circle，分别代表三角形、正方形和圆形。这三个派生类分别实现了 getArea() 函数来计算对应图形的面积。在主函数中进行了简单的测试。

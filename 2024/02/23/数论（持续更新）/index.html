数论（持续更新）

符号一览

同余符号（≡）

两个整数a，b，如果a mod m = b mod m则称a，b对于模m同余
记作a ≡ b ( mod m )

MOD

mod，要与一般的%相区分
mod意为模意义下结果一定为正
%是一种运算，结果可以为负

sigma（Σ ）

求和符号

求积符号Π

求积符号，这个公式代表的就是n的阶乘

μ莫比乌斯函数

代表了莫比乌斯函数。

φ欧拉函数

在数论中代表欧拉函数

定义：小于n的正整数中与n互质的数的数目

|整除符号

整除符号：x|y，表示x整除y，即x是y的因数

gcd（x，y）

代表x和y的最大公约数，也可写作（x，y）

lcm（x，y）

代表x和y的最小公倍数，也可写作[X,Y];

整除

若整数n除以整数d的余数为 0记为d ∣ n

a∣b, b∣c⇒a∣c

若b ∣ a，且b ∣ c，则b ∣ ( a ± c )

a ∣ b ⇒ a ∣ b *c ， c 为任意的整数。

a ∣ b , a ∣ c ⇒ a ∣ k b ± l c k 与 l 均为任意的整数。

a ∣ b , b ∣ a ⇒ a = ± b

a = k b ± c ⇒a,b 的公因数与 b , c 的公因数完全相同

若 a ∣ b* c，且 a与 c 互质，则 a ∣ b

素数

（伯特兰 — 切比雪夫定理）

若整数 n > 3 n ，则至少存在一个质数 p ，符合 n < p < 2 n − 2 。另一个稍弱说法是：对于所有大于 1 的整数 n ，至少存在一个质数 p ，符合 n < p < 2 n 。

素数定理

小于x的素数的个数近似等于x/ln(x)…

求素数可以详见素数筛博客

最大公约数与最小公倍数

设a b =不全为零, 同时整除a b的整数称为他们的公约数，显然a b的公约数只有有限多个，我们将其中最大的一个称为a b的最大公约数表示
当( a , b ) = 1，我们称a与b互质(互素)
gcd(a,b)=gcd(a−b,b)

进一步得到

gcd(a,b)=gcd(b,amodb)

lcm(a,b) gcd(a,b)=ab*

需要注意的是，如果在代码中使用，不要直接写成（a*b）/gcd（a，b）而是先算a/gcd（a，b）在乘b，放置超出数据范围。

上述也就是欧几里得算法。

扩展欧几里得定理

扩展欧几里得算法实际上就是对于ax+by=gcd(a,b)，一定有一组整数解x，y使其成立

裴蜀定理

裴蜀定理的概念及证明

因为翻译版本的不同，这个定理可能还会被叫做贝祖定理。

裴蜀定理是这样被描述的：

∀a,b∈Z,∃(x,y)∈Z

满足：

ax+by=gcd(a,b)

文字描述是这样的：对于任意的整数a,b都存在一对整数x,y使得ax+by=gcd(a,b)成立。

裴蜀定理的应用

裴蜀定理：

ax+by=gcd(a,b)

那么可以推出：如果一个数m满足：ax+by=m，那么这个m一定是gcd(a,b)的倍数。

那么对于一个经典方程ax+by=1，利用裴蜀定理，我们有：gcd(a,b)=1，即a,b一定互质。

对于关于x,y的方程

ax+by=c:若c mod gcd(a,b)!=0,则该方程无整数解

证明:根据定义,

a,b mod gcd(a,b)=0

∴若有整数解,

c mod gcd(a,b)=0

∴若c mod gcd(a,b)！=0,

则该方程无整数解证毕

如果说是有解的话，那么c一定是gcd(a,b)的倍数，因此我们只要求解ax+by=gcd(a,b)的解，

最后答案x,y也一定可以变成这个答案的倍数关系

同余

同余：数论中的重要概念。给定一个正整数m，如果两个整数a和b满足a-b能够被m整除，即(a-b)/m得到一个整数，那么就称整数a与b对模m同余，记作a≡b(mod m)。对模m同余是整数的一个等价关系

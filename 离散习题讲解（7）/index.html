离散习题讲解（7）

她来听我的演唱会……

问题：

判断下列每个函数是否是从 $ \mathbb{R} $ 到 $ \mathbb{R} $ 的双射（bijection）。

a) $ f(x) = 2x + 1 $
b) $ f(x) = x^2 + 1 $
c) $ f(x) = x^3 $
d) $ f(x) = \frac{x^2 + 1}{x^2 + 2} $

解答与解释：

a) $ f(x) = 2x + 1 $

  • 单射 (Injective): 我们可以验证该函数是单射。假设 $ f(x_1) = f(x_2) $，即 $ 2x_1 + 1 = 2x_2 + 1 $。简化后得到 $ x_1 = x_2 $，说明该函数是单射。
  • 满射 (Surjective): 对于任意的 $ y \in \mathbb{R} $，我们要找到一个 $ x \in \mathbb{R} $ 使得 $ f(x) = y $。解 $ 2x + 1 = y $，得到 $ x = \frac{y - 1}{2} $。由于 $ x $ 可以为任意实数，因此该函数是满射。
  • 结论: 因为 $ f(x) = 2x + 1 $ 是单射且满射，所以它是双射。
  • 答案： 是的，双射。

b) $ f(x) = x^2 + 1 $

  • 单射 (Injective): 如果 $ f(x_1) = f(x_2) $，即 $ x_1^2 + 1 = x_2^2 + 1 $，简化后得到 $ x_1^2 = x_2^2 $，这意味着 $ x_1 = x_2 $ 或 $ x_1 = -x_2 $。因此，该函数不是单射。
  • 满射 (Surjective): 对于任意的 $ y \in \mathbb{R} $，我们需要找到一个 $ x \in \mathbb{R} $ 使得 $ f(x) = y $。但是，$ f(x) = x^2 + 1 $ 的值始终大于或等于 1，因此不能覆盖 $ \mathbb{R} $ 中小于 1 的所有值。所以它不是满射。
  • 结论: 由于该函数既不是单射也不是满射，因此它不是双射。
  • 答案： 不是的，双射。

c) $ f(x) = x^3 $

  • 单射 (Injective): 假设 $ f(x_1) = f(x_2) $，即 $ x_1^3 = x_2^3 $，那么 $ x_1 = x_2 $（由于立方函数是单调的）。所以该函数是单射。
  • 满射 (Surjective): 对于任意的 $ y \in \mathbb{R} $，我们要找到一个 $ x \in \mathbb{R} $ 使得 $ f(x) = y $。解 $ x^3 = y $，得到 $ x = \sqrt[3]{y} $。由于 $ \sqrt[3]{y} $ 可以是任意实数，因此该函数是满射。
  • 结论: 因为该函数是单射且满射，所以它是双射。
  • 答案： 是的，双射。

d) $ f(x) = \frac{x^2 + 1}{x^2 + 2} $

  • 单射 (Injective): 假设 $ f(x_1) = f(x_2) $，即 $ \frac{x_1^2 + 1}{x_1^2 + 2} = \frac{x_2^2 + 1}{x_2^2 + 2} $。可以通过交叉相乘并化简得到 $ x_1 = x_2 $ 或 $ x_1 = -x_2 $，说明该函数不是单射。
  • 满射 (Surjective): 对于任意的 $ y \in \mathbb{R} $，我们需要找到一个 $ x \in \mathbb{R} $ 使得 $ f(x) = y $。然而，$ f(x) = \frac{x^2 + 1}{x^2 + 2} $ 的取值范围始终在 $ (0.5, 1) $ 之间，不能覆盖整个实数集，因此它不是满射。
  • 结论: 该函数既不是单射也不是满射，因此它不是双射。
  • 答案： 不是的，双射。

总结：

  • a) 是的，双射。
  • b) 不是的，双射。
  • c) 是的，双射。
  • d) 不是的，双射。

问题：

证明：如果 $ x $ 是一个实数，那么 $ \lfloor -x \rfloor = -\lceil x \rceil $ 且 $ \lceil -x \rceil = -\lfloor x \rfloor $。

解答与解释：

我们需要证明两个等式：

  1. $ \lfloor -x \rfloor = -\lceil x \rceil $
  2. $ \lceil -x \rceil = -\lfloor x \rfloor $

第一个等式：$ \lfloor -x \rfloor = -\lceil x \rceil $

  • 定义：

    • $ \lfloor y \rfloor $ 表示不大于 $ y $ 的最大整数。
    • $ \lceil y \rceil $ 表示不小于 $ y $ 的最小整数。

  • 设 $ x = n + e $，其中 $ n $ 是整数，$ 0 \leq e < 1 $ 是 $ x $ 的小数部分。

  • 对于 $ -x = -n - e $，我们有：

    • $ \lfloor -x \rfloor = \lfloor -n - e \rfloor $。
    • 由于 $ -n - e $ 的小数部分是负的，且不超过 $ -n $，所以：
      • $ \lfloor -n - e \rfloor = -n - 1 $。

  • 接下来，计算 $ \lceil x \rceil $：

    • $ \lceil x \rceil = \lceil n + e \rceil = n + 1 $（因为 $ 0 \leq e < 1 $，所以向上取整得到 $ n + 1 $）。

  • 因此，$ -\lceil x \rceil = -(n + 1) = -n - 1 $，与 $ \lfloor -x \rfloor $ 相同。

  • 结论：$ \lfloor -x \rfloor = -\lceil x \rceil $。

第二个等式：$ \lceil -x \rceil = -\lfloor x \rfloor $

  • 设 $ x = n + e $，其中 $ n $ 是整数，$ 0 \leq e < 1 $ 是 $ x $ 的小数部分。

  • 对于 $ -x = -n - e $，我们有：

    • $ \lceil -x \rceil = \lceil -n - e \rceil $。
    • 由于 $ -n - e $ 的小数部分小于 0，且其绝对值不超过 1，所以：
      • $ \lceil -n - e \rceil = -n $（向上取整）。

  • 接下来，计算 $ \lfloor x \rfloor $：

    • $ \lfloor x \rfloor = \lfloor n + e \rfloor = n $（因为 $ 0 \leq e < 1 $，所以向下取整得到 $ n $）。

  • 因此，$ -\lfloor x \rfloor = -n $，与 $ \lceil -x \rceil $ 相同。

  • 结论：$ \lceil -x \rceil = -\lfloor x \rfloor $。

总结：

我们已证明了两个等式：

  1. $ \lfloor -x \rfloor = -\lceil x \rceil $
  2. $ \lceil -x \rceil = -\lfloor x \rfloor $

因此，结论得证。

问题：

给定函数 $ f(x) = x^2 + 1 $ 和 $ g(x) = x + 2 $，它们是从 $ \mathbb{R} $ 到 $ \mathbb{R} $ 的函数。求 $ f \circ g $ 和 $ g \circ f $。

解答与解释：

我们分别求出 $ f \circ g $ 和 $ g \circ f $ 的表达式。

1. $ f \circ g $ 的计算：

$ f \circ g $ 表示 $ f(g(x)) $，即首先应用 $ g(x) $，然后将结果代入 $ f(x) $。

  • $ g(x) = x + 2 $
  • $ f(x) = x^2 + 1 $

所以，

$
f(g(x)) = f(x + 2)
$

将 $ x + 2 $ 代入 $ f(x) = x^2 + 1 $ 中：

$
f(x + 2) = (x + 2)^2 + 1
$

展开：

$
(x + 2)^2 + 1 = x^2 + 4x + 4 + 1 = x^2 + 4x + 5
$

因此，$ f \circ g(x) = x^2 + 4x + 5 $。

2. $ g \circ f $ 的计算：

$ g \circ f $ 表示 $ g(f(x)) $，即首先应用 $ f(x) $，然后将结果代入 $ g(x) $。

  • $ f(x) = x^2 + 1 $
  • $ g(x) = x + 2 $

所以，

$
g(f(x)) = g(x^2 + 1)
$

将 $ x^2 + 1 $ 代入 $ g(x) = x + 2 $ 中：

$
g(x^2 + 1) = (x^2 + 1) + 2 = x^2 + 3
$

因此，$ g \circ f(x) = x^2 + 3 $。

结论：

  • $ f \circ g(x) = x^2 + 4x + 5 $
  • $ g \circ f(x) = x^2 + 3 $

注意，$ f \circ g(x) $ 和 $ g \circ f(x) $ 不是相等的，这证明了函数的复合运算不是交换的，即 $ f \circ g \neq g \circ f $。

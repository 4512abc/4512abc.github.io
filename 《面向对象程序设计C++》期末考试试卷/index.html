面向对象程序设计C++期末考试试卷

01、下列关于 C++ 函数的叙述中，正确的是 ( C )。

A) 每个函数至少要具有一个参数
B) 每个函数都必须返回一个值
C) 函数在被调用之前必须先声明
D) 函数不能自己调用自己

正确答案：C) 函数在被调用之前必须先声明。
在 C++ 中，函数可以在调用之前声明，也可以在调用之后定义。但编译器在解析函数调用时，需要知道函数的声明或原型。

02、下列关于类和对象的叙述中，错误的是 ( A )。

A) 一个类只能有一个对象
B) 对象是类的具体实例
C) 类是对某一类对象的抽象
D) 类和对象的关系是一种数据类型与变量的关系

错误答案：A) 一个类只能有一个对象。
在 C++ 中，一个类可以有多个对象实例化。类是模板，对象是类的具体实现。

03、在 C++ 中，用于实现运行时多态性的是 ( D )。

A) 内联函数
B) 重载函数
C) 模板函数
D) 虚函数

正确答案：D) 虚函数。
C++ 中，通过使用 虚函数 和继承，可以实现运行时多态性。虚函数允许子类重写父类的函数，从而在运行时决定调用哪个函数版本。

04、下列关于[运算符]重载的叙述中，正确的是 ( B )。

A) 通过运算符重载，可以定义新的运算符
B) 有的运算符只能作为成员函数重载
C) 若重载运算符 +，则相应的运算符函数名是 +
D) 重载二元运算符时，必须声明两个形参

正确答案：B) 有的运算符只能作为成员函数重载。
C++ 中有一些运算符（如赋值运算符 = 和下标运算符 []）只能重载为类的成员函数。

05、对于语句 cout << x << endl;，错误的描述是 ( D )。

A) “cout” 是一个输出流对象
B) “endl” 的作用是输出回车换行
C) “x” 是一个变量
D) “<<” 称作提取运算符

正确答案：D) “<<” 称为插入运算符，不是提取运算符。
它用于将数据插入到输出流中，而提取运算符是 >>，用于从输入流中提取数据。

06、有如下类声明，则类 MyDERIVED 中保护的数据成员和成员函数的个数是 ( B )。

A) 4
B) 3
C) 2
D) 1

正确答案：B) 3
在类 MyDERIVED 中，保护的数据成员和成员函数的个数是 3：
1. 成员变量 j 是保护的（protected）。
2. 从基类继承的 MyBASE::set() 和 MyBASE::get() 被保护性继承，因此它们变为保护成员。
总计：3 个保护成员（1 个变量 + 2 个函数）。

07、已知在一个类体中包含如下函数原型：

Volume operator-(Volume) const;
下列关于这个函数的叙述中，错误的是 ( B )。

A) 这是运算符 - 的重载运算符函数
B) 这个函数所重载的运算符是一个一元运算符
C) 这是一个成员函数
D) 这个函数不改变类的任何数据成员的值

正确答案：B) 这个函数所重载的运算符是一个一元运算符。
解析：这个函数重载的是减法运算符 -，但它是一个二元运算符，因为它需要两个操作数：调用对象和参数对象。

08、执行如下的程序段后，输出结果是 ( C )。

在构造函数 XB 的下列定义中，正确的是 ( B )。

A) XB::XB(int a, int b): x(a), y(b) { }
B) XB::XB(int a, int b): XA(a), y(b) { }
C) XB::XB(int a, int b): x(a), XB(b) { }
D) XB::XB(int a, int b): XA(a), XB(b) { }

正确答案：B) XB::XB(int a, int b): XA(a), y(b) { }
解析：在派生类 XB 的构造函数中，需要首先调用基类 XA 的构造函数来初始化基类的部分。因此，应该使用 XA(a) 来调用基类构造函数。然后，初始化派生类成员 y，可以使用初始化列表 y(b)。

15、在表达式 x + $y * z$ 中，+ 是作为成员函数重载的运算符，* 是作为非成员函数重载的运算符。则 operator+ 有 __ 个参数，operator* 有 __ 参数。( C )

A) 2、2
B) 2、1
C) 1、2
D) 1、1

正确答案：C) 1、2
解析：在表达式中，operator+ 作为成员函数重载时，它的第一个参数是隐式的 this 指针，表示操作符的左操作数。因此 operator+ 只有 1 个参数。而 operator* 作为非成员函数重载时，它需要显式接收两个参数（操作符两侧的对象），因此 operator* 有 2 个参数。

16、应在下列程序划线处填入的正确语句是 ( C )。

A) fun();
B) Base.fun();
C) Base::fun();
D) Base->fun();

正确答案：C) Base::fun();
解析：在派生类中，如果需要显式调用基类的同名函数，可以使用作用域解析运算符 :: 来指定基类的名称和函数名。选项 A 只是调用当前类的函数，选项 B 和 D 的语法不正确。因此，正确答案是 Base::fun();。

17、面向对象程序设计有四个主要特点，即抽象、封装、继承和多态性。

18、非成员函数应声明为类的 friend 函数才能访问这个类的 private 成员。

19、派生类中的成员不能直接访问基类中的 private 成员。

20、在用 class 定义一个类时，数据成员和成员函数的默认访问权限是 private。

21、运算符重载函数可能是类的 友元 函数，也可以是类的 成员 函数，还可以是普通函数。

22、用流对象的成员函数控制输出格式时，用于设置字段宽度的流成员函数的名称是 width()，与之作用相同的控制符名称是 setw()。

23、含有纯虚函数的类称为 抽象类。

解析：

  • 17：面向对象的四个主要特点是：抽象、封装、继承和多态性。
  • 18：要使非成员函数访问类的 private 成员，可以将该函数声明为类的 friend 函数。
  • 19：派生类中的成员函数无法直接访问基类中的 private 成员，只有基类的成员函数可以访问。
  • 20：在 class 定义的类中，默认的数据成员和成员函数的访问权限是 private。
  • 21：运算符重载函数可以作为友元函数、成员函数或普通函数。
  • 22：流成员函数 width() 和控制符 setw() 都用于设置输出的字段宽度。
  • 23：含有纯虚函数的类是抽象类，无法直接实例化，只能用于派生类。
